# 🚀 БЫСТРЫЙ СТАРТ

## Что это?

Профессиональный сайт-портфолио фрилансера, специализирующегося на AI и автоматизации для бизнеса.

## Запуск за 3 шага

```bash
# 1. Установить зависимости
npm install

# 2. Запустить dev-сервер
npm run dev

# 3. Открыть в браузере
# http://localhost:5173
```

## Что дальше?

### Обновить контент
📝 Откройте `src/content/siteContent.js` - там все тексты сайта

### Добавить кейсы
📊 Отредактируйте `projects.json` - добавьте свои проекты

### Настроить контакты
✉️ Обновите контактные данные в:
- `src/pages/Contact.jsx`
- `src/components/ContactForm.jsx`

### Собрать для продакшена
```bash
npm run build
```
