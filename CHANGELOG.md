# Changelog

Все значимые изменения в проекте документируются в этом файле.

## [2.0.0] - 2026-03-21

### Трансформация: Enterprise AI Showcase → Freelance Personal Brand

Полная трансформация сайта из корпоративной витрины AI-технологий в профессиональный сайт фрилансера.

#### Добавлено

**Фаза 0: Подготовка для внешнего аудита**
- Prerender для 8 публичных маршрутов (/, /cases, /services, /about, /contact, /cases/:slug)
- Content manifest (/content-manifest.json) с полной структурой сайта
- Audit page (/audit) для ручного анализа структуры
- sitemap.xml с 9 URL
- robots.txt с правилами для поисковых роботов
- Семантическая HTML-разметка в prerendered контенте
- Уникальные meta-теги для каждой страницы
- Canonical links для всех маршрутов
- Скрипты: prerender.js, generate-manifest.js, generate-sitemap.js

**Фаза 1: Реструктуризация**
- Новая навигация: Главная, Кейсы, Услуги, Обо мне, Контакты
- 5 новых страниц: Cases, Services, About, Contact + CSS
- Полностью переделанная главная страница (7 секций)
- Русскоязычная навигация

**Фаза 2: Кейсы и данные**
- Бизнес-ориентированная модель данных в projects.json
- Компонент CaseCard с бизнес-фокусом
- Детальные страницы кейсов (/cases/:slug)
- Фильтры по категориям
- Технические детали в аккордеоне (вторично)

**Фаза 3: Личный бренд**
- Страница "Обо мне" с сильным позиционированием
- Компонент ContactForm с минимальными полями
- FAQ блок (4 вопроса)
- Блок "Почему со мной работают" (4 преимущества)
- CTA на всех ключевых страницах
- .env.example для конфигурации

**Фаза 4: Оптимизация**
- Динамический импорт Mermaid.js (ленивая загрузка)
- Улучшенные meta-теги (Open Graph, Twitter Card)
- Динамические title для всех страниц
- Динамические meta descriptions для кейсов

**Фаза 5: Контент**
- Централизованный файл контента (siteContent.js)
- 3 варианта Hero
- 3 варианта About
- 4 услуги с полным описанием
- 2 варианта "Как работаем"
- 8 вопросов FAQ
- 5 типов CTA блоков

**Фаза 6: Усиление позиционирования**
- Hero с четкой специализацией: "Автоматизация поддержки и обработки заявок"
- Блок "С кем я работаю" (4 сегмента клиентов)
- Блок "С какими задачами я помогаю" (5 проблем → решений)
- Блок "С чем я не работаю" (отсечение нецелевых клиентов)
- Обновлены все meta-теги с четким позиционированием
- Обновлены заголовки всех страниц

**Фаза 7: Усиление личного бренда**
- Добавлено имя: "Александр"
- Добавлена роль: "Фриланс-специалист по AI и автоматизации"
- Визуальный аватар с инициалом "А" (градиент)
- Переписан блок "Обо мне" (живой текст от первого лица)
- Добавлен блок "Мой подход"
- Усилен блок "Почему со мной работают" (5 преимуществ с деталями)
- Добавлен блок "Как можно начать работу" (4 формата)
- Убраны безличные формулировки по всему сайту


#### Изменено

- Основной бандл уменьшен с 694 кБ до 203 кБ (71% снижение)
- Увеличен до 216.86 кБ после добавления Audit page (+13.86 кБ)
- Mermaid.js вынесен в отдельный чанк (497 кБ, lazy loaded)
- Навигация переведена на русский язык
- Фокус смещен с технологий на бизнес-ценность
- Кейсы переименованы в клиентские названия
- Header обновлен с CTA кнопкой
- Home.jsx полностью переделан как лендинг (12 блоков)
- Позиционирование уточнено: от широкого к специализированному
- Тон изменен: от безличного к личному (от первого лица)
- Добавлен визуальный персональный элемент (аватар)
- Dockerfile обновлен: Node 18 → Node 20 (для jsdom)

#### Удалено

- Developer-first презентация
- Технические детали с первого экрана
- Копирование промптов
- Favorites функционал
- Playground элементы
- Безличные формулировки ("Помогаю бизнесу...")
- Широкие неконкретные формулировки
- 18 дублирующихся отчетных файлов (консолидированы в один)

#### Оптимизировано

- Начальная загрузка: 65.22 кБ (gzipped)
- Время сборки: 7.42с
- SEO метрики (динамические meta-теги)
- Mobile UX
- Accessibility
- Структура документации (1 полный отчет вместо 18 файлов)
